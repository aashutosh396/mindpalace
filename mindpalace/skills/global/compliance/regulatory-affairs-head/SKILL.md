Regulatory strategy, submission management, global market access for medical devices.

## Regulatory strategy
Gather product info (intended use, classification/risk, technology, target markets/timeline) → identify regulations per market → determine optimal pathway → timeline + milestones → resources + budget → risks + mitigation → stakeholder approval.

**FDA pathway selection:**
| Factor | 510(k) | De Novo | PMA |
|---|---|---|---|
| Predicate | Yes | No | N/A |
| Risk | Low-Mod | Low-Mod | High |
| Clinical data | Usually not | May be | Required |
| Review time | 90 days | 150 days | 180 days |
| User fee (~) | $22K | $135K | $440K |
| Best for | Me-too | Novel low-risk | High-risk novel |

## FDA submissions
**510(k):** confirm predicate (K-number) + substantial-equivalence argument + no new intended use/tech → pre-sub (Q-Sub) if needed → compile package (cover letter, admin, device description, intended use, SE comparison table, performance testing, biocompatibility ISO 10993-1, software IEC 62304 + cybersecurity, labeling, summary) → RTA check → eCopy → submit via ESG + fee → respond to AI/RTA within deadlines.
SE table columns: characteristic · predicate · subject device · same? · notes (new technological characteristics must not raise new safety/effectiveness questions — back with bench data).
**PMA:** confirm Class III / no predicate → IDE clinical study → pre-sub → 5-volume submission (admin/description/manufacturing · nonclinical · clinical · labeling · manufacturing+sterilization) → address deficiencies → facility inspection.
Common deficiencies: weak SE comparison · incomplete test protocols · missing biocompatibility endpoints · inadequate software hazard analysis · inconsistent labeling claims.

## EU MDR CE marking
Confirm classification (Annex VIII) → conformity route by class (I self-declaration · IIa/IIb NB · III full NB) → engage Notified Body → compile Annex II technical documentation (device description + UDI · design/manufacturing · GSPR checklist mapped to evidence · benefit-risk + ISO 14971 risk file · V&V · post-market surveillance plan · Annex XIV clinical evaluation report) → ISO 13485 QMS → submit to NB → coordinate audit → CE certificate + Declaration of Conformity + EUDAMED registration.
GSPR row: ref · requirement · standard/guidance · evidence document · status.
NB selection: designation scope, capacity/timeline, technology experience, geography, fee transparency, responsiveness.

## Global market access
Sequence markets to leverage evidence: Phase 1 FDA + EU (reference) → Phase 2 recognition markets (Canada MDSAP, Australia) → Phase 3 major (Japan, China) → Phase 4 emerging. Build a master technical file + localization plan; single-source the core (risk mgmt, clinical data with bridging, ISO 13485 cert) and localize labeling/IFU (translation, local symbols ISO 15223-1).

## Regulatory intelligence
Monitor FDA Federal Register (daily), device database (weekly), EU Official Journal + MDCG, NB communications, ISO/IEC (quarterly). Assess relevance → impact (timeline, resources, product changes) → compliance action plan with owners + dates → implement within effective-date window.

## KPIs
First-time approval rate >85% · on-time submission >90% · review-cycle compliance >95% · regulatory hold time <20%.

## Escalation
Submission rejection → VP Regulatory (root cause + strategy revision) · major deficiency → Director (cross-functional team) · timeline at risk → Management · safety signal → Executive (immediate containment + reporting).
