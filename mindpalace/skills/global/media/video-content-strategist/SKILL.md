# Video Content Strategist

Build a compounding video presence. Video is the highest-trust format — build for depth first, distribution second.

## Mode 1 — Strategy & channel setup (no presence yet)
1. **Niche & positioning** — too broad is the #1 mistake. Test: describe your ideal subscriber in one sentence. Positioning = Who (specific audience) × What problem × What you provide (unique POV) × Why you (credibility).
2. **Content pillars** — 3-4 recurring categories; every video maps to one. (e.g., how-to tutorials = search volume; tool reviews = commercial intent; case studies = trust; opinion = shareable.)
3. **YouTube SEO** — treat as the 2nd-largest search engine. 1 primary keyword per video, in: title (first 60 chars), description (first 2 sentences), tags, spoken in first 30s. Target long-tail/problem-specific keywords for new channels, not "marketing tips".
4. **90-day launch plan** — wks 1-2 setup + first 3 videos; wks 3-6 publish 1-2/wk; wks 7-10 double down on what retains; wks 11-13 repurpose top videos into Shorts.

## Mode 2 — Script & production
**Long-form structure:** Hook (0-30s — 70%+ decide here: problem statement / counterintuitive claim / result promise) → Context (30-90s) → Body (90% of runtime; Problem→Solution→Example→Result per point; chapters for >8min) → CTA (final 60s, ONE action).

**Short-form (≤60s):** 0-3s pattern-interrupt hook (in first frame, before any title card) → 3-15s state problem/promise → 15-50s deliver value → 50-60s CTA. Captions always on (85% watch muted); 9:16 vertical; one idea per video.

## Mode 3 — Repurpose & distribute
One long-form source → 1 full YouTube video + 3-5 short clips + platform-adapted versions. Blog→video map: H2 → chapters; key stats → pull-quote B-roll; steps → tutorial segments; conclusion → short clip. Workflow: identify highest-traffic source → extract the single most compelling insight → write a ≤60s script → adapt framing/captions per platform → stagger the release.

## Proactive flags
- No hook in first 3s (retention drops 40%+ before 30s).
- Targeting broad keywords too generic to rank.
- Unsustainable upload cadence (the algorithm punishes gaps).
- No chapters on videos >6min (chapters show in search → CTR).
- No CTA or buried CTA.
- Repurposing horizontal YouTube to Reels without 9:16 reformatting (60-80% worse reach).
