# Webinar & Virtual Event Marketing

A webinar is a funnel, not an event. Registrations are cheap; attention and action are not. Most value is won/lost in the parts people skip: promotion runway, show-up sequence, follow-up.

## Gather context (one section at a time)
- **Event**: topic + single promise to attendee, format, date/length/platform, live vs on-demand vs both.
- **Audience & goal**: who (role, awareness stage), business goal (leads/acceleration/adoption/retention/brand), conversion action after.
- **Reality check**: reachable list size + promo channels, runway, paid budget vs organic.

## Modes
1. **Plan from scratch** — lock promise + format, funnel math backward, promo plan, show-up sequence + live-to-close, segmented follow-up.
2. **Optimize/rescue** — get the actual numbers (invited → registered → showed → engaged → converted), score the funnel to find the weakest stage, fix that stage (don't rewrite the landing page when the problem is show-up).
3. **Evergreen/on-demand** — find the strongest live-to-close moment, set up registration→watch→follow-up automation, be honest about simulated-live framing.

## Funnel math (plan backward from the goal)
```
20 SQOs ÷ attendee→SQO (~10%) → 200 engaged attendees
       ÷ register→attend (~35% live) → ~570 registrations
       ÷ landing-page CVR (~40%) → ~1,425 visits
```
If the math needs more traffic than your reachable audience can supply, the plan is broken before it starts — fix goal/format/budget now.

## The five stages
1. **Registration** — landing page leads with outcome + takeaway (not agenda); host name + face; minimum fields; visitor-timezone time.
2. **Show-up** (where most webinars fail) — immediate confirmation + one-click calendar add; reminders at 1 week / 1 day / 1 hour / "we're live" (1-hour + live drive the most); give a reason to attend LIVE (Q&A, live-only resource, build-along); pre-event engagement.
3. **Live engagement** — hook in first 2-3 min (promise + quick win); interaction every 5-10 min; teach something genuinely useful; watch the drop-off curve.
4. **Live-to-close** — earn the right with real value first; transition explicitly and confidently; time-bound attendee-only offer; one clear repeated CTA.
5. **Follow-up** (converts more than the live event for B2B) — segment: attended+engaged (strike now), attended/no action (recap + key takeaway + soft CTA), registered/no-show (often the biggest segment — "sorry we missed you" + recording + same offer), later-watchers (recap + on-demand CTA). Send recording within 24h.

## What to avoid
Promoting topic not takeaway · one reminder email · no reason to attend live · 45 min teach then hard pitch with no transition · treating no-shows as lost · optimizing registrations as the headline metric · recording sent a week late · 8 form fields.

## Proactive flags
<3 reminders → attendance suffers. No live-only incentive → registrants "watch later" and never convert. Registration page leads with agenda → reframe to outcome. No no-show follow-up → biggest convertible segment ignored. Funnel goal > reachable audience → mathematically broken. Offer with no value delivered first → reads as bait-and-switch. >60 min for cold audience → drop-off risk.
