# Interview System Designer

Create structured interview loops, standardize question quality, and keep hiring signal consistent across interviewers.

## Core capabilities
- Interview loop planning by role and level
- Round-by-round focus and timing recommendations
- Suggested question sets by round type
- Scoring + calibration frameworks
- Bias-reduction and process consistency guidance

## Recommended workflow
1. Generate a baseline loop plan for the role + level.
2. Align each round to role-specific competencies (objectives explicit, non-overlapping).
3. Validate scoring rubric consistency with panel leads.
4. Review for bias controls before rollout.
5. Recalibrate quarterly using quality-of-hire / hiring-outcome data.

## Best practices
1. Keep round objectives explicit and non-overlapping — each round measures distinct competencies.
2. Require evidence for every score recommendation.
3. Use the same baseline rubric across comparable roles.
4. Revisit loop design based on quality-of-hire outcomes.

## Common pitfalls
- Overweighting one round while ignoring other competency signals
- Unstructured interviews without standardized scoring
- Skipping calibration sessions for interviewers
- Changing the hiring bar without documenting the rationale
