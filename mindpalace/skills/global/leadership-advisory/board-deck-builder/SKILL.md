# Board Deck Builder

Build decks that tell a story, not just show data. Every section: **Headline → Data → Narrative → Ask/Next**. Fill metric gaps with explicit placeholders — never invent numbers.

## Deck structure (standard order)

1. **Executive summary (CEO)** — exactly 3 sentences: state of business, biggest thing this period, where next quarter.
2. **Key metrics dashboard** — 6–8 metrics max, as a table: Metric / This / Last / Target / Status. Use metrics the board tracks.
3. **Financial update (CFO)** — P&L summary, cash + runway, burn-multiple trend (3 quarters), variance to plan (one sentence per variance), next-quarter forecast.
4. **Revenue & pipeline (CRO)** — ARR waterfall (start→new→expansion→churn→end), NRR + logo churn, pipeline by stage in $, forecast with confidence level, top 3 deals (name/amount/date/risk).
5. **Product (CPO)** — shipped (3–5 bullets, user impact each), shipping next (with dates), PMF signal (NPS, DAU/MAU, adoption), one key research learning. No feature lists without impact.
6. **Growth & marketing (CMO)** — CAC by channel, pipeline contribution by channel ($), brand metrics, what's working/cut/tested.
7. **Engineering (CTO)** — velocity trend, tech-debt ratio + plan, uptime/incidents/cost, security one line. Short unless material.
8. **Team & people (CHRO)** — headcount vs plan, hiring pipeline + time-to-fill, attrition (regrettable vs not), engagement, key hires/open roles.
9. **Risk & security (CISO)** — control status, compliance certs + deadlines, incidents, top 3 risks + mitigation.
10. **Strategic outlook (CEO)** — ranked next-quarter priorities, decisions needed, specific asks.
11. **Appendix** — model, full pipeline, cohort charts, case studies.

## Narrative framework (4-act)

1. Where we said we'd be. 2. Where we actually are (honest). 3. Why the gap (one cause per variance, not excuses). 4. What we're doing (specific, dated). Board should know the key message by slide 3.

## Delivering bad news

State it plainly → own the cause → show you understand it (analyze the pattern) → present the fix (specific, dated) → update the forecast (with assumptions). Never bury it, never lead with good news to soften it, never blame "market conditions" alone.

## Common mistakes → fixes

Too many slides (>25) → cut · metrics without targets → add target+status · no narrative → add story · burying bad news → lead with it · vague asks → person-assigned · no variance explanation → one-sentence cause · designing for reader not the room → it's presented, must work spoken.

## Cadence

Quarterly: full deck 20–30 slides, sent 48h ahead. Monthly (early stage): condensed 8–12 slides. Fundraising: open with market/vision, close with ask.
