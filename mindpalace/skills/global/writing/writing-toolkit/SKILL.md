# Writing Toolkit

Every writing problem has a name. Name it precisely and the fix becomes obvious. The most common mistake is treating a structural problem as a prose problem, or a character problem as a plot problem — applying the wrong tool while the real issue goes unaddressed.

## Process

1. Read the situation (paste passage or describe what isn't working). Confirm the content type and core problem in one sentence.
2. Diagnose the cluster, present the 2–4 best-fit techniques, and run on selection using the context already gathered.

## Problem clusters and techniques

**Fiction craft** — storytelling machinery: `character-development`, `plot-structure`, `scene-construction`, `dialogue`, `worldbuilding`, `arc-design`, `pov`.

**Continuity & consistency** — story contradicts itself (timeline, character knowledge, world-rules, physical continuity): `inconsistency-audit`.

**Editing & revision** — material exists, needs work: `restructure` (reorder), `tone-alignment`, `line-editing` (sentence-level), `prose-elevation`, `voice-consistency`.

**Professional & non-fiction** — real-world document to write or improve: `report`, `copy` (marketing/landing/ad), `technical` (docs/API/user guide), `argument` (make the case), `executive-summary` (1-page brief).

**Rhetoric & audience** — calibrate for a reader or surface rhetorical moves: `audience-calibration`, `rhetoric`.

## Routing by output type

- Need a diagnosis → audit skills (`inconsistency-audit`, `pov`, `scene-construction`)
- Need a rebuild → design skills (`plot-structure`, `arc-design`, `character-development`)
- Need a revision → editing skills (`restructure`, `line-editing`, `tone-alignment`)
- Need a produced piece → production skills (`report`, `executive-summary`, `copy`, `technical`)
- Need a rhetorical read → analysis skills (`rhetoric`, `argument`)

## Notes

- Don't confuse problem types. A flat character is almost never a prose problem — it's a character-design problem; line-editing won't fix it. Diagnose the layer first.
- Structure before style. Fix structural problems before prose elevation or line editing — rewriting sentences in the wrong order is wasted work.
- If the situation spans types (e.g., a scene with both dialogue and structural problems), offer both and start with the highest-leverage one.
