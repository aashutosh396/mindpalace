# Creativity Toolkit

Applies creative thinking to any challenge. Confirm the actual challenge in one sentence, then run the right single method or sequence several for comprehensive exploration.

## Which method fits

| You need to... | Method |
|---|---|
| Question whether the problem is framed right | Assumption Excavator |
| Escape an idea you keep returning to | Lateral Thinking |
| See all options before choosing | Alternatives / Concept Fan |
| Evaluate one idea fairly | Plus/Minus/Interesting |
| Think from multiple angles | Six Hats |
| Think from other people's positions | Other Perspectives |
| Explore without premature judgment | Water Logic |
| Break fixation with a random jolt | Random Entry |
| Use an absurd premise as a springboard | Provocation |
| Inventory every relevant factor | Consider All Factors |
| Big challenge, apply everything | Full sequence (below) |

## The methods

- **Assumption Excavator** — List every assumption in the framing (what's possible/wanted/fixed/who's involved); challenge each; find which are load-bearing. Output: assumption inventory + the one reframe that most changes the problem.
- **Lateral Thinking** — Name the dominant idea, set it aside, generate 5+ directions that don't involve it. Output: dominant idea named + 5+ different directions + the 1–2 worth developing.
- **Alternatives (APC)** — Quota of 5+ alternatives before any evaluation, then compare. The first idea crowds out better ones when evaluation starts too early.
- **Concept Fan** — Step back a level (what concept does this serve?), generate ways to serve it; step back again (what purpose?). Output: three levels — solutions, concepts, purposes.
- **Plus/Minus/Interesting (PMI)** — Genuine positives, genuine negatives, and what's noteworthy regardless. Antidote to snap judgment.
- **Six Hats** — Sequence pure hats: White (facts), Red (feelings), Black (risks), Yellow (benefits), Green (alternatives), Blue (process). No mixing.
- **Other Perspectives (OPS)** — For each affected person: their goal, their view of the problem, their worry, their good outcome. Structured empathy, not sympathy.
- **Water Logic** — Follow where ideas flow ("where does this go?") rather than judge ("is this true?"). Map the territory before evaluating.
- **Random Entry** — Apply a random word/object; force connections, even absurd ones; keep the 1–2 worth pursuing.
- **Provocation (Po)** — State an impossible premise; don't evaluate it — ask what the world looks like if true, then extract real-world ideas.
- **Consider All Factors (CAF)** — Inventory every factor (stakeholders, risks, resources, timing, dependencies, second-order effects); surface the ones about to be ignored.

## Full brainstorm sequence

For big challenges: (1) Assumption Excavator to clear framing → (2) Lateral Thinking to escape the dominant idea → (3) Concept Fan / Alternatives to expand options → (4) Six Hats / PMI to evaluate the best → (5) Consider All Factors to catch what's missed. Make the connection explicit between tools. Close with synthesis: what the tools together reveal that none showed alone, the single most important direction, and the concrete next step.
