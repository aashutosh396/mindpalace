# Decision Toolkit

Applies structured decision thinking to any choice. First confirm the actual choice and intended outcome in one sentence, then route to the tool that fits.

## Which tool fits

| You need to... | Tool |
|---|---|
| Make sure you're seeing all the real options | Option Mapping |
| Compare options against weighted criteria | Criteria Weighting |
| Stress-test a decision by imagining it failed | Premortem Analysis |
| Calibrate how much process this decision deserves | Reversibility Analysis |

Routing: about to choose between 2 options → Option Mapping first (the option space is almost always too narrow). Options visible, need systematic comparison → Criteria Weighting. Leaning toward a direction, want to pressure-test → Premortem. Unsure how much effort this deserves → Reversibility. When unclear → Option Mapping.

## Option Mapping

Counter the false dichotomy: the first two options are rarely all that exists. Generate across three levels — (1) direct solutions, (2) alternative framings of the problem that suggest different solutions, (3) options that combine or transcend the initial set. Quota: at least 5 options before any evaluation.
**Output:** Option inventory (5+), including reframes, with brief descriptions. No evaluation yet.

## Criteria Weighting

(1) List what actually matters. (2) Weight each criterion 1–5. (3) Score each option per criterion 1–5. (4) Compute weighted scores. (5) Check the winner against intuition — if numbers say X but gut says Y, that gap is information, not a flaw.
**Output:** Weighted decision matrix, ranked options, and whether the quantitative result matches or challenges intuition.

## Premortem Analysis

It's 12 months from now and the decision failed badly. Write the failure story — specific mechanisms, not "it didn't work." Generate 5+ distinct failure paths. Then: which are most probable? Most damaging? Mitigable before committing?
**Output:** 5+ failure paths ranked by probability and severity; mitigation for each high-priority mode; adjusted recommendation.

## Reversibility Analysis

Two-way-door decisions (easily reversed) should be made fast — deliberation costs more than being wrong. One-way-door decisions deserve thorough analysis. Classify: what would undoing it take, how fast, at what cost? Many "irreversible" decisions aren't; many casual ones are more binding than they look.
**Output:** Reversibility score (1–5), what undoing requires, and the recommended level of process.
